## 🥮 진행방식

### 📌 스터디 전

- 정해진 분량을 읽고 각자 내용 정리
  - 정리 형식: 자유
  - 권장사항: 관련된 추가 자료 찾아보기 / 질문할 내용 정리해오기 👍🏻
- 스터디 당일 오후 9시까지 정리한 내용 공유 ⭐️⭐️
  - 공유 방식: 주차별 깃허브 이슈에 댓글로 자료 url 남기기
  - 불참시에도 자료 공유는 필수 ⭐️

### 📌 스터디 당일

- 랜덤으로 2명의 발표자 선정
- 발표자는 자유 형식으로 발표
- 이야기하고 싶은 내용에 대해 토론, 질문 및 공유
